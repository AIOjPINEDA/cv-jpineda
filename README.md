# CV Jaime Pineda

Public web CV built with Astro and deployed with GitHub Pages.

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deployment

This repository deploys to GitHub Pages using `.github/workflows/deploy.yml`.

Public URL after Pages is enabled:

```text
https://aiojpineda.github.io/cv-jpineda/
```
